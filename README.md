# LV-Client-Sample

# Requirements
1. **gradle**
~~~
$ brew install gradle
~~~

2. **Java Version 1.8**


### Build
~~~
$ gradle build
~~~

### Test Client Run
~~~
$ cd build/libs
$ java -jar LV-Client-Sample.jar -m 0
~~~

### Command line Run

~~~
$ cd build/libs
$ java -jar LV-Client-Sample.jar -t "blahblash" -n 3 -th 2
$ cat clues.txt
~~~
